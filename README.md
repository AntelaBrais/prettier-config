# `@antelab/prettier-config`

> Antelab [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
# npm
$ npm i -D @antelab/prettier-config

# yarn
$ yarn add --dev @antelab/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@antelab/prettier-config"
}
```
